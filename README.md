# Pizza Recipes Android App 🍕

Une application Android de recettes de pizzas développée en Java, permettant aux utilisateurs de parcourir différentes recettes avec leurs ingrédients, descriptions et étapes de préparation.

## 🍕 Recettes disponibles

### BARBECUE CHICKEN PIZZA
- **Prix** : 3.0 €
- **Temps** : 35 min
- Pizza au poulet barbecue simple et rapide

### SPINACH PIZZA
- **Prix** : 2.0 €
- **Temps** : 25 min
- **Ingrédients** : Sauce Alfredo, épinards, tomates, fromage
- **Description** : Pizza crémeuse et légère aux légumes
- **Étapes** :
  - STEP 1: Étaler la pâte
  - STEP 2: Ajouter la garniture

##  Fonctionnalités

- ✅ **Splash Screen** : Écran de démarrage avec animation de 2 secondes
- ✅ **Liste des pizzas** : Affichage avec noms, prix et temps de préparation
- ✅ **Détail d'une pizza** : Informations complètes (ingrédients, description, étapes)
- ✅ **Design responsive** : Interface adaptée aux différents écrans

## 🛠 Technologies utilisées

- **Langage** : Java
- **IDE** : Android Studio
- **SDK minimum** : API 21 (Android 5.0 Lollipop)
- **Architecture** : Pattern Singleton pour la gestion des données
- **Stockage** : En mémoire (sans base de données)
